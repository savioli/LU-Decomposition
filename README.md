
# LU Decomposition

LU Decomposition in C++

## Example Output

	Matrix System

	5 1 -2 10 
	3 -9.4 1.8 22 
	1 2.2 4.6 10 


	Solution of Ly = b

	y0 = 10
	y1 = 16
	y2 = 11.2


	Solution of Ux = b

	x0 = 3
	x1 = -1
	x2 = 2
